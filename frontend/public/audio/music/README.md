## Swapping in longer tracks later

These jingles are 3–5 seconds each — they loop, but the seam will be
audible. To upgrade to proper looping BGM, replace any of these files
with a longer CC0 / CC-BY OGG keeping the same filename:

- Kenney's larger "Music Pack" (longer, atmospheric): <https://kenney.nl/assets>
- Pixabay Music (royalty-free): <https://pixabay.com/music/>
- OpenGameArt (CC0 / CC-BY): <https://opengameart.org>

Recommended specs: OGG Vorbis, ~96 kbps mono or 128 kbps stereo, 30–90s,
loop-ready (no fade-out at the end).
